# ⚡ Life Agent — Personal AI App

A personal AI-powered Progressive Web App 
built for students to manage their daily life.

## 🌐 Live Demo
https://resilient-rugelach-e0d042.netlify.app

## ✅ Features
- 📧 Reads Gmail automatically using Gmail API
- 🤖 AI extracts deadlines and tasks from emails
- 🔔 Daily push notifications at 7am, 12pm, 8pm
- 📋 Tracks internships and hackathons
- 🌿 Morning wellness and streak tracker
- 🐍 Python & DSA 8-week placement roadmap
- 📱 Installable on phone like a real app (PWA)
- 🔒 All data private — stored on your device only

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Claude AI API (Anthropic)
- Gmail API (Google)
- Service Workers (Push Notifications)
- Progressive Web App (PWA)
- Deployed on Netlify
